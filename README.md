# MVP-Machine-Learning-Analytics
 Projeto de Clusterização — WineQT MVP de Machine Learning para segmentação de vinhos por atributos físico-químicos.   Comparação entre K-Means, DBSCAN e Hierárquico com avaliação via Silhouette Score.   Insights de negócio: segmentação premium vs entrada, acidez e outliers especiais.   Pipeline completo, documentado e reprodutível em Google Colab.
